1. **Clone the repository:**

   ```bash
   git clone https://github.com/coryzimmerman/radar-chart-app.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd radar-chart-app
   ```

3. **Install dependencies:**

   ```bash
   npm install
   ```

4. **Running the Project**

   ```bash
   npm start
   ```
